# CompliAI

AI-Assisted Compliance Monitoring Dashboard

**> Developed as part of my internship project in compliance automation and AI-based reporting systems.**


**Overview**

CompliAI is a lightweight web application designed to help organizations monitor compliance, identify policy gaps, and track risks through an AI-assisted dashboard. It provides modular panels for Policies, Risks, Audits, Reports, and more, offering a clear overview of compliance posture.

The system is built for rapid deployment as an internal tool or prototype for compliance automation platforms.


**Features**

- Dynamic navigation and panel-based UI layout

- Policy, Risk, and Report management panels

- Simulated API fetch requests to demonstrate data loading

- Ability to generate compliance reports with timestamp

- “Mark as Reviewed” interaction to track actioned recommendations

- Footer navigation for Privacy, Terms, and Contact information

- Modular structure for easy expansion
  

**Tech Stack**

**Frontend** - 	HTML, CSS, JavaScript

**UI Components** - 	Vanilla JS DOM manipulation

**Backend (Mock)** - 	Sample JSON + simulated fetch calls
